# SyedOS — Editable Content Model

## Purpose

This document defines all content that will be editable through the private SyedOS admin dashboard.

The public website must not depend on personal information written directly inside page components.

Profile information, career stages, projects, skills, resumes, education, experience, themes, and feature settings will be stored as structured CMS data.
